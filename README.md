# School Management API

A simple Node.js API built using Express.js and MySQL to manage school data.  
It allows users to add schools and retrieve a list of schools sorted by proximity to a given location.

---

## 🚀 Features

- Add a new school
- Get list of schools sorted by distance
- MySQL database integration
- RESTful API design

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MySQL
- dotenv
